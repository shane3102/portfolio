Aplikacja ankietowa o zmiennej prezentacji pytań. Projekt tworzony w ramach pracy inżynierskiej. Posiada strukturę monolityczną. Aplikacja składa się z frontendu (Angular 14), backendu (Spring Boot 2.5.13 + Java 11) oraz bazy danych (PostgreSQL).

Głównym zamysłem aplikacji (poza wypełnianiem i tworzeniem ankiet) była możliwość przeanalizownaia wpływu sposobu wyświetlania pytań ankietowych na wypełnianie ankiety przez użytkownika. Aplikacja w swojej głównej funkcjonalności generuje 4 losowe ankiety spośród pytań dodanych przez użytkowników:

  • dobry UX, estetyczny UI
  • zły UX, estetyczny UI
  • dobry UX, nieestetyczny UI
  • zły UX, nieestetyczny UI
    • Podczas wypełniania ankiety przez użytkownika śledzony jest czas potrzebny na odpowiedź na pytanie ankietowe oraz wypełnienie całej ankiety względem sposobu prezentacji. Dane te są dostępne do pobrania i analizy. Wyczerpujący opis funkcjonalności aplikacji dostępny jest w samej pracy inżynierskiej dostępnej w linkach.
